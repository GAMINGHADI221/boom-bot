# হোস্টিং গাইড (Render.com এবং Railway.app)

আপনার Python Flask প্রজেক্টটি সহজে Render বা Railway-তে ডেপ্লয় করার জন্য প্রয়োজনীয় ফাইলগুলো তৈরি করা হয়েছে। নিচে বাংলায় কিভাবে গিটহাবে আপলোড করে ডেপ্লয় করবেন তা বুঝিয়ে দেওয়া হলো:

---

## ১. গিটহাবে (GitHub) প্রজেক্ট আপলোড করা
যে কোনো হোস্টিংয়ে ডেপ্লয় করার আগে আপনাকে কোডগুলো গিটহাবে আপলোড করতে হবে।

1. আপনার গিটহাব অ্যাকাউন্টে লগইন করে একটি নতুন **Private** বা **Public** রিপোজিটরি তৈরি করুন।
2. আপনার প্রজেক্ট ডিরেক্টরিতে গিট ইনিশিয়ালাইজ করে কোড পুশ করুন:
   ```bash
   git init
   git add .
   git commit -m "setup deployment configs"
   git branch -M main
   git remote add origin <your-github-repo-url>
   git push -u origin main
   ```

---

## ২. Render.com এ ডেপ্লয় করার নিয়ম

Render-এ ডেপ্লয় করা খুবই সহজ কারণ আমরা `render.yaml` তৈরি করেছি যা সব কনফিগারেশন স্বয়ংক্রিয়ভাবে হ্যান্ডেল করবে।

1. [Render.com](https://render.com) এ লগইন করুন।
2. ড্যাশবোর্ড থেকে **New +** বাটনে ক্লিক করে **Blueprints** সিলেক্ট করুন।
3. আপনার গিটহাব অ্যাকাউন্ট কানেক্ট করে এই প্রজেক্টের রিপোজিটরি সিলেক্ট করুন।
4. Render আপনার `render.yaml` ফাইলটি ডিটেক্ট করবে। এরপর **Apply** বাটনে ক্লিক করুন।
5. Render নিজে থেকেই ডকার ইমেজ বিল্ড করে আপনার প্রজেক্টটি চালু করে দেবে। 
6. ডেপ্লয় সম্পন্ন হলে Render আপনাকে একটি লাইভ URL দিবে (যেমন: `https://jxe-room-spam-web.onrender.com`)।

---

## ৩. Railway.app এ ডেপ্লয় করার নিয়ম

Railway ডকারফাইল (`Dockerfile`) দেখলে স্বয়ংক্রিয়ভাবে ডকার রানটাইমে প্রজেক্টটি রান করে।

1. [Railway.app](https://railway.app) এ লগইন করুন।
2. **New Project**-এ ক্লিক করুন।
3. **Deploy from GitHub repo** সিলেক্ট করে আপনার প্রজেক্টের রিপোজিটরি সিলেক্ট করুন।
4. **Deploy Now** বাটনে ক্লিক করুন।
5. ডেপ্লয়মেন্ট শুরু হলে, সেটিংস (Settings) ট্যাবে গিয়ে **Generate Domain**-এ ক্লিক করে আপনার লাইভ ডোমেন/লিঙ্কটি জেনারেট করে নিন।

---

## প্রজেক্টের গুরুত্বপূর্ণ ফাইলের কাজ:
* **Dockerfile:** এটি রেলওয়ে ও রেন্ডারকে বলে দেয় প্রজেক্টে পাইথন ইন্সটল করে `app.py` রান করতে।
* **requirements.txt:** প্রজেক্ট রান করার জন্য প্রয়োজনীয় সব পাইথন প্যাকেজ এখানে ডিফাইন করা আছে, যা হোস্টিং সার্ভার স্বয়ংক্রিয়ভাবে ডাউনলোড করে নেবে।
